Beable Y AI Linux Compilation
1) make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- distclean
2) make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- defconfig
3) make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- menuconfig
4) make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- -j8
5) How to install modules to the SD Card after compilation
   sudo make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu-INSTALL_MOD_PATH=/media/nestor/rootfs modules_install
6) Copy the kernel after compilation to SD Card
cp arch/arm64/boot/Image.gz /media/nestor/BOOT/
7) Copy DTBO files
cp arch/arm64/boot/dts/ti/*.dtbo /media/nestor/BOOT/overlays/

8) Copy DTB files. 
cp arch/arm64/boot/dts/ti/k3-am67a-beagley-ai.dtb /media/nestor/BOOT/ti

9) Where the DTS source for Beagle Y AI is stored. 
ls arch/arm64/boot/dts/ti/k3-am67a-beagley-ai.dts

10) Compile dts file only 
make ARCH=arm64 CROSS_COMPILE=aarch64-linux-gnu- -j$(nproc) dtbs

11) Deploy Manually
scp arch/arm64/boot/dts/ti/k3-am67a-beagley-ai.dtb debian@192.168.8.198:/boot/firmware/ti

Beagle Y AI Buildroot
1) make ARCH=arm64 menuconfig
2) make ARCH=arm64 -j$(nproc)
